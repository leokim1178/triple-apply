# Triple-Mileage-Service

## Git Convention

---

- **Commit message**

  - 🎉 INIT : 프로젝트 초기 환경 구축
  - ✨ ADD : 기능 신규 개발
  - 🎨 MODIFY : 기존 기능 수정
  - ⚡️ REFACTOR : 기존 기능 개선
  - 🐛 FIX : 버그 수정
  - 🔥 REMOVE : 불필요한 로직,폴더,파일 제거
  - git issue 관련 내용 상세
    - Resolve: #이슈 넘버 (이슈를 해결함)
    - Ref: #이슈 넘버 (이슈 참고)
    - Relate: #이슈 넘버 (관련된 이슈)
  - 예시

  ```bash
  ✨ ADD / 00기능을 구현하였습니다.

  - 00기능을 구현하였습니다.
  - ~~ 코드를 수정하였습니다.

  Resolve: #00
  Ref: #00
  Relate: #00
  ```

- **Branch name**
  - **feat** → **develop**
  - **develop** → **main**
